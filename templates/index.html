Mastitis Detection System for Dairy Cows

It utilizes machine learning to detect mastitis early in dairy cows by analyzing clinical data like temperature, milk color, and udder hardness. It employs SVM, Gradient Boosting, and Random Forest, with a Voting Classifier for high accuracy. The system enhances dairy farm efficiency by reducing economic losses, improving milk quality, and minimizing antibiotic use.

Human Resource Management System

Managing employee records, payroll, leave tracking, and attendance manually can lead to inefficiencies and errors. This project aims to digitalize HR operations by developing a comprehensive system to streamline these processes. The system provides a user-friendly interface with core HR functionalities, including employee management, payroll processing, attendance tracking, and department administration. By leveraging modern web technologies and APIs, the project ensures real-time updates, seamless data synchronization, and improved accuracy in HR operations.

Diabetes Prediction System

It leverages machine learning to predict the likelihood of diabetes based on various health parameters such as glucose level, BMI, blood pressure, and age. It employs models like Support Vector Machine (SVM), Random Forest, and Logistic Regression, optimizing accuracy through hyperparameter tuning. The system aims to assist healthcare professionals in early detection and intervention, improving patient outcomes and reducing long-term complications.

Hybrid System for Predicting Mastitis and FMD in Dairy Cows

It develops a hybrid model to detect Mastitis and Foot-and-Mouth Disease (FMD) in dairy cows using image processing and machine learning. Udder images will be analyzed for Mastitis, while mouth and foot images will help identify FMD. The system enhances detection accuracy through image preprocessing, augmentation, and machine learning models trained on labeled datasets from veterinary hospitals and farms. By enabling early, accurate, and non-invasive diagnosis, this solution helps farmers reduce economic losses, improve herd health, and ensure better milk quality, promoting sustainable livestock management.

Namma Kudla : A Travel Website

Namma Kudla is a dynamic and user-friendly travel website designed to showcase the rich culture, heritage, and tourist attractions of Mangaluru. Built using React.js, Node.js, HTML, CSS, and JavaScript, the platform provides an interactive and seamless experience for travelers seeking information about the best places to visit, local cuisine, events, and accommodations in the region.This project aims to enhance travel experiences by offering a digital platform that simplifies trip planning and provides valuable insights into Namma Kudla vibrant culture and attractions.
